Demande explicite d'Anne-Sophie

Un assistant RH interne capable de redonner les process de l'entreprise aux gestionnaires RH des agences quand ils posent des questions.

Cas d'usage proposés pour la journée

Dev RH / Formation

Cas d'usageStatutComment on le fait en formation
Assistant RH Clinitex interne ◐ On construit un Project Claude "RH Clinitex" avec les process en knowledge. Les RH agences posent leurs questions dedans. Ce n'est pas un agent autonome (pas de MCP / API), mais ça joue le même rôle en usage quotidien.
Tri et analyse de CV ✓ Upload des CV + fiche de poste, Claude note et justifie la shortlist.
Réponses aux mails RH agences ✓ Via le Project RH, réponses cadrées depuis les process.
Supports pédagogiques ✓ Artifacts + Skill pptx / docx. Sur un vrai sujet de formation Clinitex.
Modules e-learning ◐ Scripts, contenus, QCM générés. Export vers plateforme LMS = hors scope.
Conventions et certifications de réalisation ✓ Project avec trames Clinitex, génération en masse depuis une liste Excel.
Organisation logistique formations ✗ Réservation salles, coordination déj = hors scope (pas d'accès aux systèmes). On peut seulement générer les listes et plannings.
Préparation audit Qualiopi ✓ Checklist Qualiopi, analyse d'écarts sur documents existants, rédaction assistée.
Analyse questionnaires middle management ✓ Upload des réponses, synthèse, détection de signaux faibles et patterns récurrents.
Analyse d'indicateurs RH ✓ Upload d'exports (turnover, absentéisme...), synthèse et recommandations.
Reporting obligations sociales ◐ Génération des contenus à partir d'exports. Mise à jour automatique des outils = hors scope.

Juridique

Cas d'usageStatutComment on le fait en formation
Rédaction sur trames (courriers, actes, contrats) ✓ Project Claude avec trames Clinitex en knowledge. Chaque rédaction respecte la structure existante.
Analyse de dossiers complexes multi-documents ✓ Upload de 10-30 docs dans un Project, Claude répond aux questions et produit des synthèses.
Extractions massives et analyses de données ✓ Analyse d'exports Excel volumineux, tableaux croisés, synthèses.
Veille juridique ◐ Recherche web one-shot sur un sujet. Monitoring continu = hors scope (besoin de workflow).
Chiffrage de risques ✓ Analyse d'un dossier, identification des risques, fourchettes de chiffrage justifiées.

Paie

Cas d'usageStatutComment on le fait en formation
Réponses aux questions paie des salariés / agences ✓ Project Claude avec conventions collectives + procédures paie en knowledge.
Contrôles de cohérence paie ✓ Upload d'un bulletin ou d'un export, détection d'anomalies.
Rédaction courriers paie (STC, attestations...) ✓ Trames Clinitex, génération à partir d'un cas concret.

Cas d'usage Paie à valider / compléter via relance formulaire.

Note Agent IA — En formation Claude Pro, on ne construit pas d'agents autonomes (avec MCP, API, intégrations). On montre comment créer des Projects Claude (knowledge + prompt système) qui jouent ce rôle en usage quotidien. Pour un vrai "Agent RH Clinitex" connecté aux systèmes internes, c'est un projet dev à part.
