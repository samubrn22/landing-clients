Acte 1

Belle relation client

Le client est signé. Comment on le garde.

"J'ouvre mes mails lundi matin"

Résumer 10 mails, identifier les urgences, drafter 3 réponses. Chaque DA sur ses vrais mails.

"Je fais un contrôle qualité sur site"

Analyser 1 CQ de son agence → note, alertes, plan d'action, mail client. Sur les vrais CQ 2026.

"Je veux anticiper les problèmes"

3 CQ du même site → détecter la tendance, préparer la revue qualité. Dashboard Artifacts.

"Je dois appeler mes clients"

Claude prépare le brief avant chaque appel en mode conversation guidée. Historique + points d'attention.

Acte 2

Belle relation prospect

Le prospect n'a pas signé. Comment on convertit.

"Je prépare mon rendez-vous"

Veille sur le prospect : secteur, enjeux, concurrents. Chaque DA sur un vrai prospect de sa zone.

"Je fais l'offre commerciale"

Brief → page "Vos besoins" avec les verbatims du prospect → intégration dans le template PPT Clinitex.

"Je relance mes anciens prospects"

Ouvrir les relances M7 de sa zone → Claude priorise → script d'appel personnalisé. Excel live.

"Je surveille mon territoire"

Veille territoriale sur sa zone géo. On crée ensemble un Skill réutilisable.

Acte 3

Belle relation collaborateur

L'agent est sur le terrain. Comment on le valorise.

"J'ai un entretien pro avec un agent"

Synthèse année, points forts, objectifs, CR structuré. Être dans l'échange, pas dans la prise de notes.

"Je dois faire la gazette de mon agence"

Bullet points → gazette complète au format Clinitex. On crée un Skill + Schedule mensuel.

"Je veux surveiller mon territoire"

Veille éco territoriale : entreprises en croissance, appels d'offres, actus marché local.
